§2Plugins carry their own license

AgentCanvas follows a small-core + plugin-ecosystem model (see Contributing). Research contributions are plugins — nodesets, graph presets, and graph nodes — that live in their authors' own git repositories and carry their own license and copyright. The Apache-2.0 license here covers the framework, not third-party plugins. Plugin authors choose a license appropriate for their code and retain full authorship — which is exactly what lets a paper cite its own implementation independently.

Citing AgentCanvas Citation is decoupled from the version number — see the Versioning Policy §8 for how to cite a fixed snapshot (tag the cited version, mint a Zenodo DOI).
