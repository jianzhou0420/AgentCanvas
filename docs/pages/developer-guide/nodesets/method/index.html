Method NodeSets

Agent and method implementations. Reasoning-only nodesets consume env and foundation-model tools over wires; neural-policy baselines wrap a trained checkpoint as a single forward node. The two adapter-pipeline nodesets (policy_vla, policy_vlnce) share one design — see Adapter Pipeline for the paradigm and the VLA-vs-VLN deltas.
