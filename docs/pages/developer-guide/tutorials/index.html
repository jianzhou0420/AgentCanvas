Tutorials

Step-by-step guides for building on AgentCanvas.

Available Tutorials

  • Component Cookbook — Comprehensive, progressive tutorial covering all customizable components: nodes, nodesets, config fields, UI customization, display fields, logging, dynamic ports, server mode, graphs, composites, dataflow loops, state containers, and policy wrapping. Start here.
  • Creating a Habitat NodeSet — Wrap the Habitat-Sim environment as a NodeSet, shown both as a local nodeset (same process) and a server nodeset (separate process). Covers BaseNodeSet, ServerNodeSet, ServerApp, wire types, and the full file layout.
  • Using the Execution Log System — Add voluntary logging to custom nodes, query logs via REST API, read JSONL files, and view results in the frontend Log Viewer. Includes a standalone runnable script and a complete example nodeset.
  • Driving the Backend from a Coding Agent — Three ways to make a coding agent (Claude Code, Cursor, etc.) drive batch evals on AgentCanvas: /architect:experiment-http (legacy skill, HTTP + bash spawn), the raw MCP server (typed tools, conversational), and /architect:experiment-mcp (default skill — iter-tree bookkeeping over MCP). Side-by-side comparison and when to combine them.
