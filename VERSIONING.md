# Versioning Policy — moved

The AgentCanvas versioning policy now lives on the doc-site:

**→ https://jianzhou0420.github.io/AgentCanvas/pages/developer-guide/repo/versioning.html**

It covers SemVer numbering, tags / releases, release candidates, and the
citation axis (Zenodo DOI + `CITATION.cff`) — with the version number
**decoupled from paper milestones** (`v1.0` ships on API stability, not on
paper submission).

This file is retained only as a redirect; the doc-site page is the single
source of truth. Do not edit this file.
